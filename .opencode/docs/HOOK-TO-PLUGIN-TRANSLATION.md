# Hook -> Plugin Translation

PAI-OpenCode ports Claude Code hook concepts to OpenCode's plugin system.

## Canonical Reference

- `PAISYSTEM/PAI-TO-OPENCODE-MAPPING.md`

## Runtime Locations

- Plugins live at `~/.config/opencode/plugins/`
- Skills live at `~/.config/opencode/skills/`
- Runtime config lives in:
  - `~/.config/opencode/settings.json` (PAI-specific)
  - `~/.config/opencode/opencode.json` (OpenCode)
