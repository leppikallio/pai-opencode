# Fabric Pattern Catalog

This catalog is intentionally lightweight.

The authoritative list of installed patterns is the directory tree under:

- `Patterns/`

## List Patterns

```bash
# List pattern directories
ls -1 ~/.config/opencode/skills/Fabric/Patterns

# Inspect a specific pattern
ls -la ~/.config/opencode/skills/Fabric/Patterns/<pattern>/
```

## Pattern Files

- `Patterns/<pattern>/system.md` - system prompt for the pattern
- `Patterns/<pattern>/user.md` (optional) - user prompt fragment
- `Patterns/<pattern>/README.md` (optional) - pattern-specific notes
