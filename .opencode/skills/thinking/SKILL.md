# Thinking

This is a router skill. Use it to select a reasoning/debate workflow.

Leaf skills (load one):

- `first-principles`: root-cause decomposition and simplification
- `council`: multi-perspective debate
- `red-team`: adversarial critique and stress testing
- `be-creative`: extended ideation workflows

Common execution helpers:

- `brainstorming`: requirements exploration before creative work
- `systematic-debugging`: structured debugging workflow
- `test-driven-development`: TDD workflow for code changes
- `verification-before-completion`: enforce evidence before completion claims
