# Reference

Create IEEE-style in-text citations and reference entries.

## Inputs To Collect

- **Source type:** journal, conference, book, book chapter, website, online article/news, technical report, standard, patent, thesis, arXiv
- **Authors/organization:** full names if possible
- **Title:** article/paper/chapter/page title
- **Container:** journal/conference/book/publisher/website/publication name
- **Date:** year + month/day if applicable
- **Volume/issue/pages/DOI:** if applicable
- **URL:** for online sources
- **Access date:** for online sources (today if not provided)

If any required fields are missing, ask for them before formatting.

## Formatting Rules

- Use numbered citations in square brackets: `[1]`, `[2]`, ...
- Assign numbers in order of first appearance (do not alphabetize)
- Reuse the same number for the same source throughout
- Put citations after punctuation (period/comma) and before colons/semicolons
- Author formatting: initials then surname; 7+ authors -> `et al.`

## Output

- Provide the **in-text citation form** (`[N]` and any specific reference form if requested)
- Provide the **reference list entry** using the closest matching template from `SKILL.md`
- If producing multiple sources, output a complete `## References` section with `[1]`...`[N]`
