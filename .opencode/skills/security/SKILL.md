# Security

This is a router skill. Use it to decide which security leaf skill to load next.

Leaf skills (load one):

- `recon`: attack surface discovery, domain/IP recon
- `web-assessment`: web app security assessment and testing workflows
- `prompt-injection`: prompt injection and jailbreak testing
- `sec-updates`: security news and updates aggregation
- `annual-reports`: annual security report aggregation
- `skill-security-vetting`: scan and vet local skills for security issues

Rule of thumb:

- If you already know the right leaf skill, load it directly.
- If not, describe the target (app/system/persona) and goal; then pick a leaf skill.
