# MEMORY

This directory is source-controlled bootstrap documentation only, not the live runtime memory root for PAI/OpenCode.

- Source control contains only this README as bootstrap documentation.
- Runtime data (STATE, WORK, LEARNING, etc.) is created and managed under the installed runtime at `~/.config/opencode/MEMORY/`.
- The reflections sink is `LEARNING/REFLECTIONS/algorithm-reflections.jsonl` in installed runtime memory.
- Do not commit real memory contents from runtime back into this repository.
